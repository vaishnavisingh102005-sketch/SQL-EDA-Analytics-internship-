# SQL Data Analysis & Exploratory Data Analysis Project

## Overview
This project combines SQL-based business analysis with Python Exploratory Data Analysis (EDA) on a sales dataset containing 1200 records and 14 features.

## Project Objectives
- Analyze customer purchasing behavior
- Evaluate payment methods and revenue trends
- Identify top-selling products
- Detect sales outliers
- Generate business insights

## Technologies Used
- SQL
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Findings

### SQL Analysis
- Total Revenue: 1,264,761.96
- Top Payment Method: Credit Card
- Top Product: Chair

### EDA Insights
- Dataset Size: 1200 × 14
- Average Order Value: 1053.97
- Online transactions dominate sales
- High-value outliers exist in TotalPrice
